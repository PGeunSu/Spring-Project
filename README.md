## 📅 프로젝트 기간
22.12.14 ~ 22.12.26 (1인)

<br>

## ⌨️ 개발 환경
* OS : Mac
* Build : Gradle
* Tool : IntelliJ Ultimate , MySql WorkBench, Git
* Front-End : Html, CSS(BootStrap) , JavaSCript, Thymeleaf
* Back-End : Java(1.8.0) Spring Boot Spring Security, Spring Data JPA
* DataBase : MySQL

<br>

## 🔌 Dependcies
* Spring Boot DevTools
* Lombok
* Spring Data JPA
* Spring Security
* Spring Web
* Thymeleaf
* QueryDsl
* Spring Validaiton

## 🖱️ 기능 구현
1. 회원가입 및 로그인 
2. Form 조건 미달시 에러 메세지 출력
3. 게시판 기능
4. 상품 등록 및 수정, 삭제 기능
5. 상품 이미지 등록 기능
6. 식품 등록 및 수정, 삭제 기능
7. 상품 리스트 출력 및 검색 기능
8. 장바구니 기능
9. 주문 기능
